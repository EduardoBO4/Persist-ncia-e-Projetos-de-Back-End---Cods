Trabalho 1 - Servidor Web

Objetivo

Esta atividade tem como objetivo avaliar o entendimento e a prática de configuração de um servidor web e o acesso aos seus conteúdos estáticos.

Descrição

Desenvolver uma história com, no mínimo, três telas, utilizando imagens e textos em um Motor de Jogos (Unity ou Godot) para criar um Mini Visual Novel ou Tirinha Digital.

Requisitos

  • Mínimo de três telas: início, meio e fim da história
  • Conteúdo carregado de um servidor externo
  • URL do servidor editável via campo de entrada
  • Utilização de serviços como GitHub (formato raw) ou InfinityFree
  • História modificável apenas alterando arquivos no servidor

Entrega

  • Projeto e Build do jogo (executável)
  • Nome dos integrantes
  • Vídeo de apresentação do projeto
